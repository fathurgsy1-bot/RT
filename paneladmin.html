Ikon HTML genz_admin_panel_auth HTML Buka Saya telah menyatukan Halaman Login, Pendaftaran, dan Panel Admin dalam satu file HTML agar transisinya mulus (tanpa pindah halaman/loading). ⚠️ Penting untuk Dipahami: Halaman web statis (HTML/JS) tidak bisa secara langsung dan aman menulis atau membaca data dari Google Spreadsheet. Jika dilakukan langsung di HTML, password dan sistem database Anda akan sangat mudah diretas. Sebagai solusi profesional dan aman, kita menggunakan Google Apps Script sebagai "Server Backend". Sistem pendaftaran akan mengirim data dari web ke Apps Script, lalu Apps Script akan men-enkripsi password tersebut menggunakan algoritma SHA-256 sebelum menyimpannya ke Spreadsheet. Berikut adalah panduan lengkap cara menghubungkan HTML di atas ke Spreadsheet Anda: Langkah 1: Siapkan Kolom Spreadsheet Anda Buka link spreadsheet Anda (https://docs.google.com/spreadsheets/...), lalu pastikan di Sheet pertama (misalnya bernama Sheet1), buat kolom pada baris pertama persis seperti ini: A1: Nama Lengkap B1: Username C1: Email D1: Password (Enkripsi) Langkah 2: Memasang Google Apps Script Di dalam Spreadsheet Anda, klik menu Ekstensi > Apps Script. Hapus semua kode yang ada (biasanya function myFunction() { }), lalu copy-paste kode di bawah ini: JavaScript // Fungsi untuk menerima pengiriman data dari Web HTML (Form Login & Register) function doPost(e) { // Ganti 'Sheet1' dengan nama sheet Anda jika berbeda var sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName("Sheet1"); var action = e.parameter.action; if (action === "register") { var nama = e.parameter.nama; var username = e.parameter.username; var email = e.parameter.email; var rawPassword = e.parameter.password; // Enkripsi password menggunakan SHA-256 var encryptedPassword = computeSHA256(rawPassword); // Cek apakah username atau email sudah ada di database var data = sheet.getDataRange().getValues(); for (var i = 1; i < data.length; i++) { if (data[i][1] === username || data[i][2] === email) { return ContentService.createTextOutput(JSON.stringify({ "status":"error", "message":"Username atau Email sudah terdaftar!" })).setMimeType(ContentService.MimeType.JSON); } } // Tambah baris baru ke Spreadsheet sheet.appendRow([nama, username, email, encryptedPassword]); return ContentService.createTextOutput(JSON.stringify({ "status":"success", "message":"Berhasil didaftarkan" })).setMimeType(ContentService.MimeType.JSON); } else if (action === "login") { var identifier = e.parameter.identifier; // Bisa username atau email var password = e.parameter.password; var encryptedPassword = computeSHA256(password); // Enkripsi input untuk dicocokkan var data = sheet.getDataRange().getValues(); // Looping mencari data: A=Nama, B=Username, C=Email, D=Password for (var i = 1; i < data.length; i++) { var dbUsername = data[i][1]; var dbEmail = data[i][2]; var dbPassword = data[i][3]; // Jika (Username ATAU Email cocok) DAN Password Enkripsi cocok if ((dbUsername === identifier || dbEmail === identifier) && dbPassword === encryptedPassword) { return ContentService.createTextOutput(JSON.stringify({ "status":"success", "message":"Login berhasil" })).setMimeType(ContentService.MimeType.JSON); } } return ContentService.createTextOutput(JSON.stringify({ "status":"error", "message":"Username/Email atau Password salah!" })).setMimeType(ContentService.MimeType.JSON); } } // Fungsi Bantuan: Algoritma Enkripsi SHA-256 function computeSHA256(input) { var rawHash = Utilities.computeDigest(Utilities.DigestAlgorithm.SHA_256, input); var txtHash = ''; for (var j = 0; j < rawHash.length; j++) { var hashVal = rawHash[j]; if (hashVal < 0) hashVal += 256; if (hashVal.toString(16).length == 1) txtHash += "0"; txtHash += hashVal.toString(16); } return txtHash; } Langkah 3: Deploy API Server Agar file HTML bisa berkomunikasi dengan script tersebut: Di halaman Apps Script, klik tombol biru Terapkan (Deploy) di pojok kanan atas > Deployment baru (New deployment).
